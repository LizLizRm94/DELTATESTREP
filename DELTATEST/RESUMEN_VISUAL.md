## ?? FLUJO DE DATOS

### Obtener Evaluación
```
Usuario hace clic en "Reporte"
    ?
ReporteEvaluacion.razor carga con ID
    ?
OnInitializedAsync() se ejecuta
    ?
ReporteEvaluacionService.ObtenerEvaluacionAsync(id)
    ?
HttpClient GET /api/evaluaciones/{id}
    ?
API: SELECT * FROM Evaluacion WHERE IdEvaluacion = id
     + INCLUDE Usuario (evaluado)
     + INCLUDE Usuario (administrador)
    ?
Retorna DetalleEvaluacionDto
    ?
GenerarHtmlReporte() transforma en HTML
    ?
Renderiza en página con @((MarkupString)htmlReporte)
    ?
Usuario ve reporte formateado
```

### Descargar como PDF
```
Usuario click: "Descargar/Imprimir"
    ?
window.print() abre diálogo nativo
    ?
Usuario: "Guardar como PDF"
    ?
CSS @media print activa
    ?
Navegador genera PDF
    ?
Archivo descargado a Descargas/
```
